Template Centralizado de Layout

Este é o modelo unificado de navegação e layout do SIGAEDU.

💡 Como usar este arquivo:

  1. Qualquer alteração visual no Menu Lateral (Sidebar), no Cabeçalho (Header) ou no Script de Navegação deve ser feita diretamente neste arquivo (unificado.html).
  2. Para propagar as alterações automaticamente para as 12 páginas do sistema, basta abrir o terminal PowerShell e rodar o comando: .\sync_layout.ps1
  3. O script irá atualizar cirurgicamente todas as páginas individuais instantaneamente, mantendo a integridade de dados e lógica interna de cada uma.

edit Alteração Fácil

Adicione, remova, edite links ou mude ícones e cores do menu centralizado aqui. O sincronizador cuida da replicação!

lock Integridade Protegida

O sincronizador atua apenas no layout de topo e menu, sem tocar nos formulários, tabelas, modais ou scripts acadêmicos de cada página!
